### 2026-06-22 跨版数据迁移工具（Web 版作中转）
- 描述：Web 版做数据导入，可作数据迁移中转站——导出 JSON → 导入 → 编辑/清理 → 导出
- 价值：数据管理有可视化工具，应用有额外价值
- 复杂度：低（导出导入已规划，加批量编辑即可）
- 候选版本：V1.0（导入）+ V1.1（批量编辑 UI）
- 风险：跨版本 schema 必须严格向下兼容（铁律 10）

### 2026-06-22 键盘宏 / 自定义快捷键
- 描述：用户可自定义快捷键绑定（如把「跳过」从 S 改成 X）
- 价值：键盘优势深化，重度用户定制
- 复杂度：低
- 候选版本：V2.0+
- 风险：默认快捷键要符合直觉，自定义是锦上添花

### 2026-06-22 多番茄钟并行（多标签页 / 多窗口）
- 描述：Web 版天然支持多标签页，可同时跑多个番茄钟（如一个工作一个学习）
- 价值：多任务并行计时
- 复杂度：中（多计时器实例 + 数据归属区分）
- 候选版本：V2.0+
- 风险：违背「专注一件事」番茄理念，需克制设计；可能与任务锁定逻辑冲突

### 2026-06-22 番茄日记 / 心情记录
- 描述：每个番茄结束后可记录三言两语 + 心情标签
- 价值：把番茄从纯计时升级为反思工具
- 复杂度：中
- 候选版本：V2.0
- 风险：可能干扰极简主义，做成可关闭的可选模块

### 2026-06-22 AI 任务建议（基于历史数据）
- 描述：基于历史专注数据，建议下一个番茄做什么
- 价值：减少决策疲劳
- 复杂度：高（需要本地小模型或云端 LLM）
- 候选版本：V3.x
- 风险：纯本地无网下 AI 能力有限，需评估本地推理可行性

### 2026-06-22 浏览器扩展（Chrome Extension）
- 描述：基于 Web 版核心逻辑做 Chrome 扩展，工具栏图标 + 弹窗计时
- 价值：技术栈可低成本扩展到浏览器扩展形态
- 复杂度：中
- 候选版本：V3.x+
- 风险：扩展有独立审核流程（Chrome Web Store），与 PWA 定位部分重叠
